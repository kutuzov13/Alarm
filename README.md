# Alarm
Будильник для пробуждения!

### Install

You need Python3 to run scripts.
Download the code from GitHub.

```
git clone 
```

### Run

```
python alarm.py
```

Input yours name: <name>

Enter the hour you want to wake up: <hours>

Enter the minute you want to wake up: <minute>

Alarm!!! Weka Up!!!
